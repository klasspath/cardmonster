# Cardmonster.de

Basierend auf [Node.js](https://nodejs.org/en/) and [Next.js](https://nextjs.org/)

# Notwendige Software

* [Node.js](https://nodejs.org/en/)

## Setup

Der Quellcode muss heruntergeladen werden. Dafür kann entweder [Git](https://git-scm.com/downloads) verwendet werden, oder der Download-Button der Repository bei GitHub.

## Programm starten

1. Quellcode herunterladen
2. In Projektordner begeben
3. Ausführen: `npm install` oder `yarn install`
4. Ausführen: `npm run dev` oder `yarn dev`
